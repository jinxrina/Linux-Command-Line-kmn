Basic Commands

pwd
pwd
Print Working Directory: Shows the current directory you are in.
Example:
$ pwd
/home/user/documents
cd
cd [directory]
Change Directory: Navigate to the specified directory.
Change to home directory:
$ cd ~
Go up one directory:
$ cd ..
Change to specific directory:
$ cd /var/www/html
Tip: Use 'cd -' to switch to the previous directory you were in.
ls
ls [options] [directory]
List Directory Contents: Display files and directories.
List with details:
$ ls -l
Show hidden files:
$ ls -a
Human-readable file sizes:
$ ls -lh
cat
cat [options] [file]
Concatenate and display file content.
View file content:
$ cat file.txt
View with line numbers:
$ cat -n file.txt
less
less [options] [file]
View file contents with pagination and scrolling.
Example:
$ less large_log_file.txt
Tip: Press 'q' to exit, '/' to search, and space bar to scroll down.
head
head [options] [file]
Display the beginning of a file (default: first 10 lines).
View first 10 lines:
$ head file.txt
View first 5 lines:
$ head -n 5 file.txt
tail
tail [options] [file]
Display the end of a file (default: last 10 lines).
View last 10 lines:
$ tail file.txt
Follow file updates in real-time:
$ tail -f /var/log/syslog
touch
touch [options] [file]
Create an empty file or update file timestamps.
Create a new file:
$ touch newfile.txt
mkdir
mkdir [options] [directory]
Create one or more directories.
Create a directory:
$ mkdir new_folder
Create nested directories:
$ mkdir -p parent/child/grandchild
cp
cp [options] source destination
Copy files and directories.
Copy a file:
$ cp file.txt backup/
Copy directory recursively:
$ cp -r folder/ backup/
mv
mv [options] source destination
Move or rename files and directories.
Rename a file:
$ mv oldname.txt newname.txt
Move a file to a directory:
$ mv file.txt /path/to/directory/
rm
rm [options] [file]
Remove files or directories.
Remove a file:
$ rm file.txt
Remove recursively with confirmation:
$ rm -ri directory/
Warning: Be careful with 'rm -rf', it deletes everything without confirmation!

File Operations

chmod
chmod [options] mode file
Change file permissions.
Make a file executable:
$ chmod +x script.sh
Set specific permissions:
$ chmod 755 file.txt
chown
chown [options] owner[:group] file
Change file owner and group.
Change owner:
$ chown user file.txt
Change owner and group:
$ chown user:group file.txt
chgrp
chgrp [options] group file
Change group ownership.
Example:
$ chgrp developers file.txt
find
find [path] [expression]
Search for files in a directory hierarchy.
Find by name:
$ find /home -name "*.txt"
Find files modified in the last 7 days:
$ find /var/log -type f -mtime -7
grep
grep [options] pattern [file...]
Search for patterns in files.
Search for text in file:
$ grep "error" logfile.txt
Case-insensitive search:
$ grep -i "warning" *.log
Recursive search with line numbers:
$ grep -rn "function" --include="*.php" /var/www/
locate
locate [options] pattern
Find files by name using a database.
Example:
$ locate "*.conf"
Tip: Run 'sudo updatedb' to update the locate database.
tar
tar [options] [archive-file] [file or directory to be archived]
Archive utility for creating and extracting file archives.
Create a tar archive:
$ tar -cvf archive.tar /path/to/directory/
Create a compressed (gzip) archive:
$ tar -czvf archive.tar.gz /path/to/directory/
Extract a tar archive:
$ tar -xvf archive.tar
gzip
gzip [options] [file...]
Compress or expand files.
Compress a file:
$ gzip file.txt
Decompress a file:
$ gzip -d file.txt.gz
zip/unzip
zip [options] [zipfile] [file...] / unzip [options] [zipfile]
Package and compress files in zip format / extract zip archives.
Create a zip archive:
$ zip -r archive.zip /path/to/directory/
Extract a zip archive:
$ unzip archive.zip

System Commands

top
top [options]
Display Linux processes in real-time with system resource usage.
Example:
$ top
Tip: Press 'q' to exit, 'h' for help, 'k' to kill a process.
htop
htop [options]
Interactive process viewer, an enhanced version of top.
Example:
$ htop
ps
ps [options]
Report a snapshot of the current processes.
Show all processes:
$ ps aux
Process tree:
$ ps -ef --forest
free
free [options]
Display amount of free and used memory in the system.
Human-readable output:
$ free -h
df
df [options]
Report file system disk space usage.
Human-readable sizes:
$ df -h
du
du [options] [file...]
Estimate file space usage.
Summarize directory usage:
$ du -sh /var/log
useradd
useradd [options] username
Create a new user account.
Create a new user:
$ sudo useradd -m -s /bin/bash newuser
passwd
passwd [options] [user]
Change user password.
Change your password:
$ passwd
Change another user's password:
$ sudo passwd username
userdel
userdel [options] username
Delete a user account.
Delete user account and home directory:
$ sudo userdel -r username
who
who [options]
Show who is logged on.
Example:
$ who
last
last [options] [user...]
Show listing of last logged in users.
Show last 10 logins:
$ last -n 10
sudo
sudo [options] command
Execute a command as another user, typically the superuser.
Run command as root:
$ sudo apt update
systemctl
systemctl [options] command [name]
Control the systemd system and service manager.
Start a service:
$ sudo systemctl start apache2
Enable service at boot:
$ sudo systemctl enable apache2
Check service status:
$ systemctl status apache2
service
service name command
Run a System V init script.
Restart a service:
$ sudo service nginx restart
crontab
crontab [options]
Schedule periodic tasks.
Edit crontab:
$ crontab -e
List scheduled jobs:
$ crontab -l
kill
kill [options] pid
Send a signal to a process, usually to terminate it.
Terminate a process:
$ kill 1234
Force kill a process:
$ kill -9 1234

Network Commands

ping
ping [options] destination
Send ICMP ECHO_REQUEST to network hosts.
Basic ping:
$ ping google.com
Limit to 5 packets:
$ ping -c 5 8.8.8.8
ip
ip [ OPTIONS ] OBJECT { COMMAND | help }
Show / manipulate routing, devices, policy routing and tunnels.
Show IP addresses:
$ ip addr show
Show routing table:
$ ip route
ifconfig
ifconfig [interface] [options]
Configure network interfaces (older alternative to 'ip').
Show all interfaces:
$ ifconfig
netstat
netstat [options]
Display network connections, routing tables, interface statistics.
Show all listening ports:
$ netstat -tuln
ss
ss [options]
Another utility to investigate sockets (replaces netstat).
Show listening TCP ports:
$ ss -lt
scp
scp [options] [[user@]src_host:]file1 [[user@]dest_host:]file2
Securely copy files between hosts on a network.
Copy from local to remote:
$ scp file.txt user@remote:/path/to/destination/
Copy from remote to local:
$ scp user@remote:/path/to/file.txt /local/path/
rsync
rsync [options] source destination
Fast, versatile file copying tool.
Sync local directories:
$ rsync -av /source/directory/ /destination/directory/
Sync with remote system:
$ rsync -avz -e ssh /local/dir/ user@remote:/path/
wget
wget [options] [URL]
Non-interactive network downloader.
Download a file:
$ wget https://example.com/file.zip
Download recursively:
$ wget -r -np https://example.com/
curl
curl [options] [URL]
Transfer data from or to a server.
GET request:
$ curl https://api.example.com/users
POST request with data:
$ curl -X POST -d "name=value" https://api.example.com/users
dig
dig [options] name [type]
DNS lookup utility.
DNS lookup:
$ dig google.com
MX record lookup:
$ dig google.com MX
nslookup
nslookup [options] [name] [server]
Query Internet name servers interactively.
Example:
$ nslookup google.com
traceroute
traceroute [options] host
Print the route packets trace to network host.
Example:
$ traceroute google.com
nmap
nmap [options] target
Network exploration tool and security / port scanner.
Scan a host:
$ nmap 192.168.1.1
Scan specific ports:
$ nmap -p 80,443 example.com
tcpdump
tcpdump [options] [expression]
Dump traffic on a network.
Capture on interface:
$ sudo tcpdump -i eth0
Filter traffic:
$ sudo tcpdump -i eth0 port 80

Advanced Commands

jobs
jobs [options]
Display status of jobs in the current session.
List all jobs:
$ jobs -l
fg
fg [job_spec]
Move job to the foreground.
Bring job 1 to foreground:
$ fg %1
bg
bg [job_spec]
Move jobs to the background.
Move job 2 to background:
$ bg %2
nice
nice [option] [command [arg]...]
Run a command with modified scheduling priority.
Run with lower priority:
$ nice -n 10 ./cpu_intensive_task
renice
renice priority [[-p] pid ...] [[-g] pgrp ...] [[-u] user ...]
Alter priority of running processes.
Example:
$ renice -n 5 -p 1234
nohup
nohup command [args] [&]
Run a command immune to hangups, with output to a non-tty.
Run in background:
$ nohup ./long_script.sh > output.log 2>&1 &
awk
awk [options] 'program' file
Pattern scanning and processing language.
Print specific columns:
$ awk '{print $1, $3}' file.txt
Sum column values:
$ awk '{sum += $1} END {print sum}' file.txt
sed
sed [options] 'script' [input-file]
Stream editor for filtering and transforming text.
Replace text:
$ sed 's/old/new/g' file.txt
Delete lines:
$ sed '/pattern/d' file.txt
xargs
xargs [options] [command [initial-arguments]]
Build and execute command lines from standard input.
With find:
$ find . -name "*.txt" | xargs grep "pattern"
expr
expr expression
Evaluate expressions.
Arithmetic:
$ expr 5 + 3
String length:
$ expr length "Hello"
sort
sort [options] [file...]
Sort lines of text files.
Sort numerically:
$ sort -n numbers.txt
Sort in reverse:
$ sort -r file.txt
uniq
uniq [options] [input [output]]
Report or omit repeated lines.
Remove duplicates:
$ sort file.txt | uniq
Count occurrences:
$ sort file.txt | uniq -c
cut
cut [options] [file...]
Remove sections from each line of files.
Extract columns:
$ cut -d',' -f1,3 data.csv
Cut characters:
$ cut -c1-5 file.txt
tr
tr [options] set1 [set2]
Translate or delete characters.
Convert to uppercase:
$ cat file.txt | tr 'a-z' 'A-Z'
Delete characters:
$ cat file.txt | tr -d ','
diff
diff [options] file1 file2
Compare files line by line.
Compare two files:
$ diff file1.txt file2.txt
Side-by-side comparison:
$ diff -y file1.txt file2.txt
wc
wc [options] [file...]
Print newline, word, and byte counts for each file.
Count lines:
$ wc -l file.txt
Count words:
$ wc -w file.txt
